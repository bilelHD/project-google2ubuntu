#Voici les commandes prédéfinies insérer à google2ubuntu

# Explications #

Pour utiliser google2ubuntu, au départ vous devez connaître quelques commandes et les sons associés à ces commandes. À chaque commande correspond un son à prononcer.


# Liste des actions #

| pause musique | mettre en pause rhythmbox |

| jouer    | Jouer une musique en l'appelant directement |

| écrire   | lancer la prise de note |

| menu     | affiche le menu |

| fermer   | ferme la fenêtre |
| monter   | simule un scroll haut |
| descendre | simule un scroll bas |
| arret    | permet de quitter google2ubuntu |
| recherche | lancer une recherche sur Google |
